Техническая документация игры Asteroids 2021. 

Проект состоит из пяти скриптов:
1. SpaceController отвечает за управление игровым объектом SpaceSheep (звездолёт):
	- метод ScorePoints - подсчёт очков при попадании астероидов.
	- метод LoseLife - счётчик жизней. 
	- метод Respawn - появление объекта после потери жизни.
	- метод HighScoreInput - сохранение лучшего результата игрока.

2. AsteroidScript отвечает за управление игровыми объектами Asteroids (большой, средний и малый астероиды):
появление, движение, разделение на несколько объектов при попадании объектов bullet (пули звездолёта).

3. Alien отвечает за управление объектом Alien (летающая тарелка):
	- движение за объектом SpaceSheep, стрельба пулями (объект BulletAlien).
	- метод NewLevel - рандомное появление объекта Alien при переходе на следующий уровень. 
4. GameManager отвечает за создание нового уровня с определённым количеством объектов. 
5. Menu отвечает за управление сценой StartMenu (меню).

Пользовательская документация игры Asteroids 2021.

Суть игры - собрать большее количество очков путём уничтожения астероидов и летающих тарелок. 
- Нужно летать на звездолёте и стрелять в парящие в космосе астероиды, которые при попадении пули распадаются на осколки. 
- Также необходимо остерегаться летающей тарелки, которая может неожиданно возникнуть и стрелять в звездолёт. 
- После попадения по звездолёту пули от тарелки или столкновения с астероидом и его осколками у звездолёта отнимается одна жизнь. 
- Управление звездолётом происходит по нажатию кнопок "вверх", "влево" и "вправо" на клавиатуре, стрелять нужно левой кнопкой мыши. 
- Лучший результат игры будет виден после проигрыша. 
- Чем меньше астероид - тем больше очков.
